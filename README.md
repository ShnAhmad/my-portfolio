## Installation

- Clone the repository:

```bash
  git clone https://github.com/ShnAhmad/my-portfolio.git
```
- Navigate to the project directory:

```bash
  cd my-portfolio
``` 
- Install dependencies:

```bash
  npm install
``` 
- Run the app:

```bash
  npm run dev
```
